# List of references

List of references

## Usage

``` r
derappp_bib
```

## Format

A list of
[RefManageR::BibEntry](https://docs.ropensci.org/RefManageR/reference/BibEntry.html)
objects

## Examples

``` r
derappp_bib[1:10]
#> [1] Agroscope, ed. _Pesticides and Intermediates Ecotoxicological Risk
#> Information System_. 2023.
#>

#> Chemistry_. 2023. <https://commonchemistry.cas.org>.
#>

#> Risk Assessment of Plant Protection Products_. R package version 0.9.2.
#> 2025. <https://agroscope-ch.github.io/derappp/>.
#> 
#> [5] National Center for Biotechnology Information. _PubChem_. 2023.
#> <https://pubchem.ncbi.nlm.nih.gov/>.
#>
```
